🏏 CricketStat

CricketStat is a Python project that simulates and analyzes cricket player performance data using NumPy and Pandas.

📊 Overview

Generates a random dataset representing cricket player statistics

Performs statistical analysis on player performance

Calculates batting metrics such as strike rate and batting average

Evaluates all-rounder performance using a custom scoring algorithm

Identifies top batsmen, best bowlers, and high strike-rate players

Selects a dream team based on overall player performance

Structured using modular programming (generator, insights, engine)
